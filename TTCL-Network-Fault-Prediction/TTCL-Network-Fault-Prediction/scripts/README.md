# 📂 Scripts Folder

## 📌 Description

This folder contains all scripts used for data processing, spatial analysis, and predictive modeling.

---

## 🗂️ Structure

* `R_scripts/` → Statistical modeling and analysis (GLM, ROC, correlation)
* `QGIS_workflow/` → GIS processing steps and project files

---

## ⚙️ Key Processes

* Data cleaning & preprocessing
* Geocoding fault locations
* Spatial analysis (terrain, buffer, correlation)
* Predictive modeling using GLM
* Model validation using ROC-AUC

---

## 🛠️ Tools Used

* R Studio (tidyverse, pROC, GGally, ggcorrplot)
* QGIS
* ArcGIS (optional advanced processing)

---

## ▶️ How to Run

1. Open R scripts in R Studio
2. Install required packages
3. Run scripts step-by-step
4. Ensure data paths match your local setup

---

## ⚠️ Notes

* File paths may need to be updated before execution
* Scripts are organized to follow the workflow of the study
