# 🔥 Burn Severity Analysis

## 🎯 Objective
To analyze and map burn severity patterns using spatial data for environmental monitoring and land degradation assessment in Kilimajaro.

---

## 🛠️ Tools Used
- :contentReference[oaicite:0]{index=0}  
- Remote sensing classification techniques  

---

## 📊 Methodology
- Spatial data preprocessing  
- Raster classification of burn severity levels  
- Map styling and cartographic design in QGIS  

---

## 🗺️ Output
This project produced a burn severity map highlighting affected areas in Kilimanjaro and intensity levels of burning.

📄 **Map Output:** Burn_Severity_Map.pdf

---

## 📌 Key Insight
The map helps identify areas with high environmental impact, supporting land management and recovery planning.
